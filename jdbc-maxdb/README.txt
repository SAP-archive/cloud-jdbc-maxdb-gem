= jdbc-maxdb

* http://jruby-extras.rubyforge.org/activerecord-jdbc-adapter/

== DESCRIPTION:

This is a MaxDB JDBC driver gem for JRuby.

Use

   require 'jdbc/maxdb'

to make the driver accessible to JDBC and ActiveRecord code running in JRuby.

=============================
Building the gem:

$> gem build jdbc-maxdb.gemspec


=============================
Installing the gem: 

$> gem install jdbc-maxdb
